Base de autobuses con listas circulares

Instrucciones para modificar la ruta

  • Para agregar una base debes insertar un nombre y una cantidad de minutos que son los que se tarda un camión en ella.
  • Para eliminar o buscar una base únicamente debes insertar su nombre.
  • Listar ya sea en orden o en reversa no requiere ningún input.

Instrucciones para crear tarjetas

  • Para crear una tarjeta debes insertar el nombre de la base inicial, la hora de inicio y la duración total del recorrido en minutos.
  • La hora debe ser un número entero entre 0 y 23.
  • La cantidad de minutos debe ser un número entero positivo.
